📋 Complete Setup Guide
From zero to live Instagram data in under 20 minutes.
1
Create a Meta Developer Account

Go to developers.facebook.com → sign in → click "Get Started" to register as a developer.

2
Create a New App → Business type

Click "Create App", choose Business. Name it "Collab Connect". This unlocks Instagram Graph API access.

3
Add Instagram Graph API Product

From Dashboard → "Add Product" → find "Instagram Graph API" → Set Up. Link a Facebook Page connected to an Instagram Business or Creator account.

4
Generate Access Token → Exchange for Long-Lived (60 days)

Tools → Graph API Explorer → select permissions → Generate. Then exchange:

GET https://graph.facebook.com/v19.0/oauth/access_token ?grant_type=fb_exchange_token &client_id={app-id} &client_secret={app-secret} &fb_exchange_token={short-lived-token}
5
Get the Instagram Business User ID

You need the IG User ID (not username) to query any creator's data:

GET https://graph.facebook.com/v19.0/me/accounts ?access_token={token} // Then for each page result: GET https://graph.facebook.com/v19.0/{page-id} ?fields=instagram_business_account &access_token={token} // Returns: { "instagram_business_account": { "id": "17841400..." } }
6
Fetch Profile + Posts → Auto-Calculate ER

The core data pipeline Collab Connect uses:

// Step A: Profile GET https://graph.instagram.com/v19.0/{ig-user-id} ?fields=id,username,name,biography,followers_count,media_count &access_token={token} // Step B: Recent 12 Posts GET https://graph.instagram.com/v19.0/{ig-user-id}/media ?fields=like_count,comments_count&limit=12 &access_token={token} // Step C: Calculate ER avgLikes = SUM(post.like_count) / 12 avgComments = SUM(post.comments_count) / 12 ER% = (avgLikes + avgComments) / followers_count × 100
7
Submit App for Advanced Access (to query any public account)

In dev mode, only accounts linked to your app are accessible. For full marketplace use, submit for Advanced Access under App Review. State your use case: influencer marketplace for brand-creator matching.

8
Paste credentials in the form above → start fetching

Once saved, every "Live" button and IG lookup uses your token automatically. All data syncs to Google Sheets if configured.

📊 Google Sheets Database — Apps Script
Every influencer pulled from Instagram auto-appends a row to your Sheet.
A
Create a Google Sheet with these headers in Row 1:
Name | Handle | Niche | Followers | Avg Likes | Avg Comments | ER% | Bio | Source | Fetched At
B
Extensions → Apps Script → paste this code:
function doPost(e) { const s = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName("Sheet1"); const d = JSON.parse(e.postData.contents); s.appendRow([d.name, d.handle, d.niche, d.followers, d.avgLikes, d.avgComments, d.er, d.bio, d.source, new Date().toISOString()]); return ContentService .createTextOutput(JSON.stringify({status:"ok"})) .setMimeType(ContentService.MimeType.JSON); } function doGet(e) { return ContentService .createTextOutput(JSON.stringify({status:"alive"})) .setMimeType(ContentService.MimeType.JSON); }

Deploy → New Deployment → Web App → Anyone

Copy the Web App URL → paste into "Google Sheets URL" in the credentials form above. Every API fetch now auto-syncs.
